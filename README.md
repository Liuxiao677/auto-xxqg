## `仅供我学习autojs使用！切勿用于违法用途，否则开发者不承担任何责任。`
## 当前最新版为v2.6.0.1

# 功能说明
### 在使用切换账户时需配置账户在账户配置页面，无需自己登陆学习强国，使用其他功能(如：默认执行)时无需配置，但必须已登录学习强国
